Working of our Project

Our project mainly aims that Enhance road safety through the implementation of intelligent vehicle vigilance systems in cab sector.

The 'Sentinel Drive' project aims to revolutionize road safety using intelligent vehicle vigilance.It integrates AI, computer vision, and sensors to create a responsive road safety ecosystem.Through features like rash driving detection, alcohol detection, and seat belt detection, it monitors driver behavior and alerts officials.
Predictive analytics anticipate hazards, and all data is saved on the cloud using deep learning. Objectives include developing an integrated automotive safety solution, monitoring real-time vehicle and road conditions, and encouraging responsible driving behavior.

Features Of Our Project

Accident Detection

This system is to identify abnormal events on the road using: various sensors like Accelerometers, IR, Vibration, GPS & GSM Technology
This helps to road reduce accidents.

RashDriving Detection

By monitoring driver is rash driving or not, rash driving also be detected by continuously Lane changing.
Accelerometer helps to detect sharp turns, Rapid changes in velocity.

Alcohol Detection

By Alcohol Detector sensor we analyze drivers condition of consumed alcohol or not. This helps to avoid cab driver to drink and drive. This also reduces complexity on road.

SeatBelt detection

Seat-belt weared or not is detected using seat-belt detector sensor,
Or we can detect using camera module by image processing.This increases chances to save driver.
